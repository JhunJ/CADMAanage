        기능 추가 방법
        
      
      
      
      
        
      왼쪽 에셋 탭의 기능 리스트에 새 기능을 등록하려면 아래 순서대로 진행하세요.
      
        
      1. 우측 기능 탭에 패널 추가
      
        
      #featureActivationContent 안에 <div id="featurePanel-새id" class="feature-panel">...</div> 형태로 HTML 블록을 추가합니다.
      
        
      2. ASSET_FEATURES에 등록
      
        
      window.ASSET_FEATURES 배열에 객체를 추가합니다.
      
        
        
          
      • id: 고유 식별자 (예: 'my-feature')
        • 
          
      • label: 리스트에 표시할 이름
        • 
          
      • panelHtmlId: 패널 요소 id (예: 'featurePanel-my-feature')
        • 
          
      • activateLabel: 활성화 버튼 문구 (선택)
        • 
          
      • onActivate: 활성화 버튼 클릭 시 실행할 함수
        • 
        
      
        
      3. 선택 시 로드
      
        
      기능 선택 시 셀렉트를 채우려면 initAssetFeatureList() 내 해당 기능 클릭 시 분기에서 필요한 로드 함수를 호출하세요.
      
        
      자세한 내용은 docs/ASSET_FEATURES.md를 참고하세요.
